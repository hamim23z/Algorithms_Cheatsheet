```
**Big Oh Notation and Growth of Functions**
O(1) < O(lgn) < O(n) < O(nlgn) < O(n^c) < O(c^n) < O(n!)
- Best: O(1)
- Worst: 0(n!), O(c^n), and O(n^c)

1. O(1): Constant Time
2. O(lgn): Logarithmic Time
3. O(n): Linear Time
4. O(nlgn): Linearithmic Time
5. O(n^c): Depends on the value of c. Usually it's quadratic time(2) or cubic time(3)
6. O(c^n): Exponential Time
7. O(n!): Factorial time
```
